Visualizing Mobile Internet Performance

Interpreting the Data

The Speedometer dataset gathers cellular internet performance data from mobile phone users around the world. We calculate aggregated Round-Trip-Times (RTTs) from pings issued to various websites by mobile clients in three major metro regions, and overlay the data on to the heatmap below.

Using the Visualization

Filter the dataset for specific parameters using the controls on the right. E.g. you can select a single carrier and a single metro region Some presets and comparisons are listed at the bottom.
