Visualizing Mobile Internet Performance

Interpreting the Data

The Speedometer dataset gathers Internet performance data from mobile phone users around the world. We calculate aggregated Round-Trip-Times (RTTs) from pings issued to various Google servers by mobile clients in three major metro regions, and overlay the data as a heatmap on to the geographic map below.

Using the Visualization

Filter the dataset for specific parameters using the controls on the right. E.g. you can select a single carrier and a single metro region. You can zoom in on specific areas to separate and compare values better.
